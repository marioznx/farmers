1) npm install -g bower - установка bower глобльно
2) npm iinstall - из папки проекта
3) bower i normalize.css jquery - утановка normalize и jquery
4) @include font("название", "путь", "начертание", "стиль"); - подключение шрифтов
5) gulp svgSprite - создание файла sprite.svg
6) @@include('includes/_header.html') - импорт файлов html
7) _header.html - название з _ чтобы не компилировать в dist
8) gulp otf2ttf - из формата otf в ttf
